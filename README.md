# Calorie Burnt Predictor

A Machine Learning project that predicts the number of calories burned during exercise using physiological and activity-related data. The model is built using XGBoost Regressor and trained on features such as age, gender, height, weight, duration, heart rate, and body temperature.

## Features

- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- Correlation analysis
- Feature engineering
- Calorie burn prediction using XGBoost Regressor
- Model evaluation and performance analysis

## Technologies Used

- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- XGBoost

## Dataset Features

- Gender
- Age
- Height
- Weight
- Duration
- Heart Rate
- Body Temperature

## Results

The model was trained and evaluated on the calorie burn dataset, achieving accurate calorie expenditure predictions based on user exercise and body metrics.
